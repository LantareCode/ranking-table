To run:
	- 	Option 1:
		>	Run the run.bat file
		>	Enter your file path when prompted for example
		>	Your results will be printed in the console.
		
	-	Option 2 (In case the .bat file does not work):		
		>	Open Command Prompt or similar
		> 	CD to the directory where the RankingTable java file are saved.
		>	Run commands: 
			1. javac RankingTable.java
			2. java RankingTable
		>	When prompted enter the path to the .txt file with the input.
		>	Your results will be printed in the console.
	
To test:
	-	Open test folder
	-	Run test.bat and follow the instructions on the screen to the test various input files.
